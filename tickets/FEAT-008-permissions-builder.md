## Overview

Visual permissions builder UI for creating and managing ReBAC permission rules. Builds on top of the rules builder to provide intuitive permission configuration.

## Key Components

- **Visual editor**: Drag-and-drop permission rules
- **Role builder**: Create custom roles with permission sets
- **Permission templates**: Pre-built permission sets
- **Preview**: Test permissions against sample data
- **Inheritance**: Visualize permission hierarchy
- **Conflicts**: Detect and resolve permission conflicts

## Use Cases

- Create custom roles beyond built-in (owner, admin, member)
- Define resource-specific permissions
- Conditional permissions (e.g., "edit if owner")
- Temporary permissions (time-based)
- Team-level permissions
